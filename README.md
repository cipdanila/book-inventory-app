# Book Inventory App 📚

[🔗 Live Demo](https://github.io)

A professional web-based inventory display built with semantic HTML5 and advanced CSS3, featuring a modern Glassmorphism UI and a custom rating system.

## 🚀 Features
- **Modern UI:** Implements a Glassmorphism design using `backdrop-filter` and semi-transparent gradients.
- **Advanced Selectors:** Heavy use of CSS Attribute Selectors (`^=`, `~=`, `=`) for precise styling without over-relying on multiple classes.
- **Custom Rating System:** A visual rating component built entirely with CSS shapes and logical attribute targeting.
- **Interactive Elements:** Smooth scale transitions and hover effects on rating stars for improved user feedback.
- **Responsive Layout:** Optimized with `overflow-x: auto` to ensure data readability on mobile devices.

## 🛠️ Tech Stack
- **HTML5:** Semantic table structure (`thead`, `tbody`) and accessible `span` elements.
- **CSS3:** 
  - **Attribute Selectors:** Targeted styling based on specific class values.
  - **Flexbox:** Used for perfect alignment of status badges.
  - **Visual Effects:** Linear gradients, `box-shadow` (inset and outer), and `backdrop-filter`.

## 💡 Key Learnings
1. **Logic with Selectors:** Using `[class~="three"]` to style multiple children based on a parent's secondary class.
2. **Data-Driven Styling:** Mimicking dynamic behavior (like book status colors) using only CSS logic.
3. **UI Polishing:** Implementing "pill-shaped" badges and internal shadows to create depth and a professional feel.
